Summer 2024

Rate My Professor

The Rate My Professor project is a platform where students can access reviews on professors, enabling more informed course and instructor selections. The application is built with Next.js and TailwindCSS for a dynamic and responsive frontend. To enhance user experience, the project utilizes a Retrieval-Augmented Generation (RAG) pipeline with the Gemini API, making reviews more contextually relevant, and leverages Pinecone as a vector database for optimized search functionality. This tech stack empowers users with comprehensive, easily accessible insights into their professors.

BIMA
Freelance

BIMA is a comprehensive construction budget and inventory management system designed to streamline and manage logistics, financial tracking, and inventory oversight. This powerful combination of technologies enables BIMA to deliver a responsive and dynamic solution tailored for the construction industry.
